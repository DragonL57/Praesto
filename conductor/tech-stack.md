# Core Tech Stack
- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **AI Integration:** Vercel AI SDK
- **Styling:** Tailwind CSS, shadcn/ui
- **Database:** Neon Serverless Postgres (via Drizzle ORM)
- **Auth:** Auth.js (NextAuth.js)
- **Storage:** Vercel Blob

# AI Tools & Integrations
- **Web Search:** Tavily & DuckDuckGo integration for real-time information.
- **Calendar:** Google Calendar API integration for schedule management.
- **Information Retrieval:** Website content scraper and YouTube transcript tool.
- **Weather:** Real-time weather forecasting tool.

# Development & Quality
- **Linting/Formatting:** Biome (Primary quality check)
- **Animations:** framer-motion
